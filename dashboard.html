Click Upload button. Follow the wizard step-by-step.

Start with low cost templates to build reputation before charging.

Upload Mastery

  • ZIP Structure: Include index.html in root folder
  • File Size: Keep under 10MB (compress images)
  • Cover Image: 1200×630px PNG/JPG for best quality
  • Preview Media: Use GIF to showcase animations
  • Description: Include keywords, features, installation steps
  • Pricing: Start competitive, increase with reviews
